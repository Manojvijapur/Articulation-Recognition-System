Articulation Recognition System

-->This is the project based on the voice commands.
-->At first, we need to say the password throw our voice, then it gets logged into the the personal voice assistant.
-->And it greets us with irrespective of times.
-->It gets the results according to wikipedia.
-->After that you can play music, search in google,youtube,whatsapp,spotify and so on, through our voice command.
-->If the result you are searching is not there, it shows result not found message and speaks the same.

Following are the steps for executing the project,

-->At first we need python interpreter, pycharm projects, visual studio.
-->Next we need to install some packages.
-->Like pyttsx3, speech recognition, wikipedia, web browser and start executing.
